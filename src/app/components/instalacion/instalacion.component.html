Hoy aprenderemos a instalar Angular usando el Angular CLI paso a paso.

  • Paso 1:

    Instalar NodeJS en su ultima versión, para eso entraremos a su web oficial y descargaremos el instalador más actualizado Nosotros en este momento estamos utilizando la versión 10.8.0 Node 10.8.0. Luego instalar en disco local C: .

  • Paso 2:

    Instalar la última versión de TypeScript TypeScript-Instalación , para eso abrir CMD o gitBach y pegar el comando.

  • Paso 3:

    Instalar Angular-CLI en su útlima versión Angular-CLI Instalación

  • Paso 4:

    Instalar Visual Studio Code VS-Code Instalación

  • Paso 5:

    Instalar Google Chrome, es sumamente importante tener este navegador, ya que tiene una de las mejores consolas de navegador actualmente. Para acceder a ella una vez ya este instalado Google Chrome presione la tecla F12 para acceder a ella.

    Esto debería ser suficiente para comenzar.
