Declaración de variables

Existen varios tipos de variables y TypeScript nos deja declarar de varias formas:

  • let: Esta anotación nos sirve para declarar cualquier tipo de variable, tal y como lo vemos en la imagen, también podemos definir los tipos de datos y asignarles un valor adecuándolo al tipo de variable.

    • También se puede declarar variables del tipo any la cual recibe cualquier valor que nosotros le asignamos, este tipo de variables es bueno usarlo al no saber qué tipo de dato recibirá nuestra función, pero siempre será una buena práctica definir nuestras variables una vez ya sepamos cual es el tipo de dato que esta recibiendo.

    También existen las variables de tipo const, estas se caracterizan por que su valor es estático y este nunca cambiara dentro del contexto en el cual se le esté declarando, es una buena práctica al nombrarlas hacerlos con MAYUSCÚLAS ya que esto nos indica que es una variable estática.

    Templates literales

    Con TypeScript podemos reemplazar nuestra forma de mostrar nuestros resultados. Para esto se implementó (` `) = ALT + 96 como una de las formas estándar y fáciles que dentro del framework podemos utilizar.

    Argumentos (obligatorios, opcionales, por defecto)

    Como bien sabemos una función trae consigo argumentos las cuales podemos definir dentro de ella, una buena practica es definir estas en el orden que vamos a visualizar.

    Primero van las obligatorias, luego las opcionales y al final las por defecto, esto hace mas entendible nuestra función y así cualquier desarrollador puede usarla con facilidad.
